# Royaviators Design System

## Brand Personality

- Premium
- Modern
- Trustworthy
- Intelligent
- Minimal
- Human
- Visionary
- Outcome-focused

## Design References

The visual direction should feel like a mix of:

- Apple
- McKinsey Digital
- IDEO
- Stripe
- Notion

Avoid:

- Generic AI imagery
- Robot visuals
- Futuristic clichés
- Stock-photo business teams
- Overused neon gradients

## Color Palette

### Primary

- Midnight: `#0F172A`
- White: `#FFFFFF`
- Light Surface: `#F8FAFC`
- Soft Border: `#E2E8F0`

### Secondary

- Slate: `#64748B`
- Deep Slate: `#334155`
- Mist: `#F1F5F9`

### Accents

- Ocean Blue: `#2563EB`
- Emerald Impact: `#10B981`
- Signal Amber: `#F59E0B`

Use accents sparingly for calls to action, diagrams, active states, and measurable impact highlights.

### Headings

- Satoshi

### Body

Recommended:

- Inter

- IBM Plex Sans

### Metrics / Numbers

Recommended:

- IBM Plex Sans
- Inter

## Layout Principles

- Large whitespace
- Clear grid system
- High contrast hero section
- Minimal but confident typography
- Large visual sections
- Wide desktop layouts
- Clean mobile stacking
- Avoid crowded sections

## Components

### Navigation

- Sticky top navigation
- Transparent over hero, solid after scroll
- CTA button: Book a Strategy Session
- Mobile hamburger menu with full-screen overlay

### Buttons

Primary:

- Dark background
- White text
- Rounded 999px
- Subtle hover elevation

Secondary:

- Transparent or light background
- Dark text
- Thin border

### Cards

- 16px to 24px radius
- Light border
- Soft shadow on hover
- Strong heading
- Short description
- Optional line icon

### Industry Cards

Each card should include:

- Icon
- Industry name
- Core challenge
- ImpactOS adaptation
- CTA: Explore solution

### Diagrams

Use interactive diagrams to show:

- ImpactOS modules
- Data and knowledge flows
- Transformation journey
- Industry-specific adaptations

### Tables

Use clean comparison tables for:

- Royaviators vs consultancy vs software vendor
- ImpactOS modules
- Service categories

### Forms

Contact form fields:

- Name
- Email
- Organization
- Industry
- Main challenge
- Project stage
- Message

## Motion Principles

- Smooth fade-in on scroll
- Subtle parallax in hero
- Animated system lines in ImpactOS diagrams
- Hover elevation for cards
- Progressive disclosure for technical content
- Gentle page transitions

Motion should feel calm, intelligent, and premium.

## Icon Direction

Use minimal line icons, preferably custom or from a refined icon system.

Icon themes:

- Aviation
- Tourism
- Ocean
- Climate
- NGO
- University
- Government
- AI Assistant
- Knowledge Hub
- Automation
- Analytics
- Connectors
- Reporting

## Mobile Design

Mobile must include:

- Sticky bottom CTA
- Clear section stacking
- Swipeable industry cards
- Accordion services and FAQs
- Simplified diagrams
- Fast load performance

## Accessibility

- Strong color contrast
- Keyboard navigability
- Visible focus states
- Descriptive link text
- Semantic headings
- Alt text for all visuals
- Reduced motion option

## Visual Direction

Use abstract visual systems:

- Flow lines
- Knowledge maps
- Modular architecture
- Soft gradients
- Clean dashboards
- Document and workflow metaphors
- Human-centered decision systems

The site should feel intelligent without looking artificial.
